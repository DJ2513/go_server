# Go server

This is a basic example of how to implement a simple server in Golang with the standard library of "net/http". It is still a work in progress.

## How to run the server

1.First you need to download the repo:

```bash
git clone https://github.com/DJ2513/go_server.git
```

2.Open the project on you IDE, I recommend `VSC` or `Cursor` and inside the terminal run this command:

```bash
# This complies and runs all go files needed
go run cmd/*.go 
```

3.On the terminal, it will apear a link you can click by combining `Control + Click`. On Mac use `command + click`.

Now you can visit the following paths:

```text
- /
- /health

More to come on the future for better examples.
```
